## Dependencies

## Installation

1. Git clone this repository
2. cd into the the client/ folder
3. Change line 55 URL to localhost:3080
4. npm install
5. npm start (starts the client)
6. cd ../ 
7. cd into the api/ folder
8. npm install
9. node index.js (start the server)
